## Files in processed/

### Update Timing

* **GitHub Actions**: Runs automatically on a weekly schedule (cron)
* **Manual**: Run `python -m scripts.fetch_peps` â†’ `python -m scripts.calculate_metrics` in this order.

### Files generated by `fetch_peps.py`

Running `python -m scripts.fetch_peps` generates the following files.

| File                   | Description                                                                               | Input                                          |
| ---------------------- | ----------------------------------------------------------------------------------------- | ---------------------------------------------- |
| `peps_metadata.csv`    | Basic information about PEPs (number, title, status, authors, etc.)                       | RST files in the GitHub python/peps repository |
| `citations.csv`        | Citation relationships between PEPs (`citing`, `cited`, `count`)                          | Same as above                                  |
| `metadata.json`        | Metadata recorded at fetch time (`fetched_at`, `checked_at`, `source_url`, `data_hashes`) | Automatically generated during processing      |
| `citation_changes.csv` | History of changes in citation relationships (added/changed/removed citations)                    | Difference from the previous `citations.csv`   |

### Files generated by `calculate_metrics.py`

Running `python -m scripts.calculate_metrics` generates the following files.
This script must be executed after `fetch_peps.py`.

| File               | Description                                                                  | Input                                                 |
| ------------------ | ---------------------------------------------------------------------------- | ----------------------------------------------------- |
| `pep_graph.pkl`    | NetworkX `DiGraph` object representing the citation graph                    | `citations.csv`, `peps_metadata.csv`, `metadata.json` |
| `node_metrics.csv` | Graph metrics for each PEP (`in_degree`, `out_degree`, `degree`, `pagerank`) | `pep_graph.pkl` (constructed internally)              |
